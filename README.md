# Password Strength Checker

I made a tool to check how strong your password it uses HTML, CSS and JavaScript. This Password Strength Checker looks at your password. Tells you how strong it is right away. It checks some security things and gives you a score on a little meter.

https://niran-cy.github.io/password-strength-checker/

## Features

* The Password Strength Checker looks at your password as you type it

* It has a meter that says if your password is Weak, Fair, Good or Strong

* It checks if your password meets all the requirements

* You can click a button to see your password

* The tool looks nice. Works well on all devices

* It tells you how strong your password is as you type

* It is small and easy to use. Does not need any other programs to work

## Validation Criteria

The Password Strength Checker uses these rules to check your password:

* Your password must be least 8 characters long

* Your password must have least one big letter, like A or B

* Your password must have least one small letter, like a or b

* Your password must have least one number, like 1 or 2

* Your password must have least one special character, like !. @

## Technologies Used

* I used HTML5 to make the structure of the tool

* I used CSS3 to make the tool look nice

* I used JavaScript to make the tool work
